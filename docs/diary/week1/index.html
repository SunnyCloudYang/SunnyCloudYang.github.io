Mon#

  • 此乃馄饨混沌开天之日，万象伊始，天地更新。
  • 换句话说，我不记得那天干啥了，大概是初始化吧（x
  • 让我们从下一天开始。

Tue#

  • 今天看完了《开端》，总体不戳（金麦是真的好看（没有说其他人不好看的意思）），不太敢评，没那实力，估计也没人想看。
  • 今天大概是，有开端没有结尾，剧也是，生活也是。

Wed#

  • 今天更是完蛋，昨天好歹有个开端，今天只能用惶惶终日来形容。
  • 早上醒来就是将近十二点了，等我慢腾腾地吃完brunch，取个快递回来，就已经下午两点了。DL依旧不想看，STM也还是没动力，Socket更是没有计划。假期开始前想做个多人在线游戏demo的计划不出意外地被搁置，一方面是想再等等蹭科协承诺的云服务器，另一方面还是有太多的事情想干，但每一件事都会占据去一部分精力和时间，最后每个都马马虎虎，只有我感觉累成🐎了但是又没干什么。
  • 大概是从小时候起我就缺乏这种安排好许多事情的能力吧。又大概，只有ddl是第一生产力吧（笑。单线程的JS有异步，但是单线程的我只能阻塞。JS得益于EventLoop，我却被它困住，也许加入光荣的赛博飞升确实是个好想法，到时候我一定会用JS 重复的生活让我感觉枯燥烦闷，没有进步更是让我上火焦心。
  • 这个状态让我想起了高三冲刺的时候，也是这种感觉，每日无所事事又焦虑于其他人看不见的努力。当时觉得这颇像一座黑暗森林，我们就是森林里的猎人，每个人都在黑暗中静静磨刀，等候猎物的出现。看不到彼此的猎人们并不知道自己的技术是否足够将猎物据为己有，只有在篝火旁相遇时短暂交流以相互探底，继而又深入到各自秘密的旅途中，如此往复，直至答案揭晓的那一刻。
    然而有时候短视的猎人们会忘记在这片大陆的其他地方，也有许许多多座黑暗森林，有些森林里甚至进化出了更为高级的狩猎方式甚至是生活方式。沾沾自喜于在自己的森林里取得的成就只会让他止步不前，可是如果没有外界偶尔闯入的旅人，谁又会主动向外张望呢。
  • 大概大多数时候我们都处于这样一片森林中吧，没有比较，少有共享，只有结果。
  • 总的来说，今天还是稍微干了点事情的。大概研究了一下异步到底怎么个事儿；写了个筛选中文并去重的小东西便于更新字体文件（原谅我并没有学会两位老师的方法orz）；看到茶园的开奖结果之后，第一次认真看了看一亩三分地的信息，占卜一下未来；另外马原没有背刺我，算是不多的欣喜了吧。
  • 昨天，键盘坏了，也许是前天，我不知道。按下退格有50%可能触发异步编辑事件，在随机延迟0.5 ~1秒后触发删除。该删的不删，不能删的咣咣删，写得我生气。所以如果文中有一些让你感觉驴唇不对卖狗肉的地方，那应该就是这家伙的功劳。
  • 我发现听着音乐不太适合再产生输出，信道隔离似乎并不好，写的东西呢，像是挂着羊头，长着马嘴。想了想，这种现象，好像叫什么，一根筋(x)。
  • 好了朋友们，咱们今天的日记就到这里吧，音乐已经开始淹没理智了。我是猫眼螺噢，我要窝沙了。喜欢看的朋友们呢，就先喜欢着吧，咱们明天见。

Thurs#

  • 今天终于搞懂了异步下载的正确方式，打算今晚或者明天再更新一下教参爬虫，应该再没什么大问题了吧。之前采用的是在最后一张图片下载完成后就显示本章下载完成，但是异步下载导致可能这之前的页面其实并没有下载完成，所以是一种“伪进度”，改进之后应该就是真正的进度了。接下来大概只剩下转换成PDF格式和重构💩山代码的问题了，也许可以在假期结束之前结束吧。
  • 另外昨天的日记充分激发了我写一个自动更新字体文件脚本的想法，并且终于接近完成了。以后不写日记的理由又少了一条 每次写这种小脚本都会让我想起那个把生活完全自动化的俄罗斯极客，虽然我和他的水平有云泥之别，我也没有那么狂热的向往，但是这种一键部署的感觉，是真的爽啊。把琐碎的事更有效率地完成，这大概就是人类使用工具的初衷吧。
  • 今天更新了课表，看了看返校的行程安排，猛然又不太意外地发现，距离返校只剩9天了。换句话说，假期已经在苟延残喘，准备向我挥手说再见了。再看看假期开始前列出的计划，又仿佛回到了假期开始的时候，shift。大概唯一坚持了一个假期的事情，就是还在坚持中午起床吧（雾。哦对了，还有好不了的咳嗽。
  • 说到这个咳嗽，是真的有些令人费解了。最初的症状出现在最后一门考试的前一天（1月7号）。当时的我还在庆幸症状轻微，没几天就只有点咳嗽了，谁知道这亿点点的咳嗽能够伴随我一个假期，就怕回到学校还在咳嗽，咳咳。
  • 记得小学时候的我，总喜欢在假期生病。一到假期就会感冒咳嗽流鼻涕，一到开学前两天就好。我和我妈都挺费解，我感觉大概是因为在家实在过于无聊，想回学校找操蛋小朋友们玩了吧。但咱也不敢直说，只能默默感谢阿瑞斯（又或是西王母）一个假期的陪伴。
  • 明天，周五了，还是学会习吧，不能一周都荒废过去啊。

Fri#

  • 转眼就到了传说中要学习的日子。
  • 我愉快地打开了珍藏已久的学习资料，看个什么呢，我徘徊着。
    今天可是说好要学习的呀，可不能再摸一天🐟，我思索着。
    那要不学点“Cutting-edge Tech”（切边柯基），我欣喜着。
    那就——《黑豹2》！终于，我心满意足。
    “真是孺子可教也啊”，我这么夸着自己，戴上我的耳朵芽儿（Earbuds）。
  • 黑豹2里说海洋中也有一个国家有Vibranium，那里的人喝了另一种Herb之后全身都变成了蓝精灵蓝了。其实看的时候倒没看成蓝精灵，倒是差点以为在看阿凡达2。不过有一说一，我还挺想看里面那个头上插着一圈羽毛的鸡冠小姐姐的，可惜虽然在她的宗族里地位不低，戏份却少得可怜，她大概是那些演员里少数带着些东亚面相特点的人。我总感觉那些海里的人们有一些亚洲血缘（或者说，有点阿三哥的模样），当然，没有比较的意思，只是陈述个人感觉哈，不一定对。为了不剧透，还是少说点细节吧。
    总的来说，2有种高中作文套模板的感觉，甚至感觉可以据此预测黑豹3的内容了：大概就是沙漠或者外太空的什么地方也有Vibranium，然后三个国家大概又打了一架，或者联合起来和其他国家打了一架，最后达成和解，和平利用Vibranium。哦对了，这个国家的人也会有一些草超能力,大概也是因为喝了一种很新的Herb。因为这个Herb呢，虽然去除了大部分的腥味，但是制作的过程中保留了一部分生草的味道，所以让人们直呼生草，最终获得了另一种生草的能力（什么烂梗）。
    不过漫威商业片套模板也不是一部两部了，就像那高中作文，套模板也许拿不了高分，但至少能求稳，已经不错了。就怕再这么拍下去，电影院里只剩下小朋友了吧（可能因为只有他们没有ddl要赶吧）。
  • 看完了阿凡黑豹2，大概真得学习了吧。我真的是这么想的，但我打开了网易云。
  • 主要还是因为这个键盘，它今天还在抽风，时不时提醒我一下它还在病着。我打会字就气得想打键盘，打会键盘就发现打了好多乱码在屏幕上，删着删着就发现它还在和我对着干。所以我放弃了，找了个不用打字的教程，边看边想，边想边走神，边走神还边唱歌（🤮yue）。
  • 最终我彻底放弃了，随便找了个设计模式的电子书看着玩。就这样，在我入睡的那一刻，时间回到了今天早上来到了周末，旁边也没有赵今麦，我也忘记了写日记。这个平淡的2月3日就像无数个2月3日和其他日子一样，被定格在失去生命的日历里，像一粒石子抛入河道，咚，再无声响。

Sat#

  • 在推迟！循环的时间在推迟！
  • 呃其实我是说，好像这样一件事情已成定局：当天的日记已经无法在当天完成，而是只能被阻塞到第二天才开始写。日记不再是日“记”，而变成了日“忆”。这也许是好事，能帮我过滤掉一些无足轻重的小事，然而就我这记性和脑袋，过滤完只怕一点渣儿都不会有，脑袋也是。
  • 今天（其实是昨天）大概又是愉快冲浪的一天。爬虫重写到一半，脑子感觉像是被鹰酱拿鳄鱼夹子夹了一样难受，于是赶紧看了一会小俞避避邪，又忍不住看了一会可（chou）爱（xiang）的捷豹。哦想起来了，应该不是被夹了，大抵是在某站随机游走时看到了鹰酱在尝试射箭，那一箭好险没射豁我的脑干。至于为什么会随机游走，难道你们的网页是不会自动跳转的吗 Σ(っ °Д °;)っ？！
  • 再后来大概就是在写前一天的日记，并且改一些细微的格式。外加干大饭，睡大觉，摆大烂。直到我下定决心关电脑睡觉时，我发现了一个问题：
    在可以强制开启深色模式的移动端浏览器里（比如Via和夸克），深色模式下的删除线无法显示！大概是强制变涩色的时候把删除线和背景一视同仁了，都改成了和背景相似的深色，可是字体还是白色。于是删除线大隐隐于文，把该说的不该说的都暴露出来了，这可怎么拌啊。我连做了三个梦也没有找到对策，只能把它放进下一轮的EventLoop。
  • 真的很奇怪，好几次问题都是在关电脑之后发现，像极了出考场的那一刻发现有道题没写，不过大抵是因为我嫌移动端调试麻烦且观感着实不佳，没有多加测试吧。
  • 古代某位大诗人曾说过：“明日复明日，明日何其多。”我想他的意思应该是说，既然明日很多，那么我们就要把明日利用好，把还不能或者还不想做的事留给明天，好好享受当下的时光罢。我觉得他说的有道理，我想的也有道理，于是欢喜地退出了浏览器，把这件事留给明天吧，不然明天没事干了可怎么办呢！

Sun#

  • 打破循环！
  • 终于，在这一周的最后一天，我跳出了循环。但是我发现了一件好玩的事情。
  • 先说别的，今天暴走了1w+步，对于常年粘在椅子上的我来说，这差不多相当于一个月的运动量了。今天沿河走了一大圈，但是没有看到社火（虽然本来也没打算看社火）。那些天鹅和鸭子们还在河面上，不过确切来说，现在大多是在冰面上。每每看到冰面上蹲着的绿头鸭🦆，我就会想起荷塘里那两只被冻在冰面上的可怜鸭子，我知道不应该嘲笑他们，毕竟他们又不用上学也不用上班，被冻住了又不会耽误什么，但是真的很搞笑可怜啊。
    立春大概只是标志着气温的一阶导数过了零点，气温依然维持在零下，所以冰面的范围还在扩大，只有水流最湍急的阶梯处还有些流动的水面。一部分鸭子们集体卧在冰面上，头朝着太阳的方向，甚至排成了一个杨辉三角。不知道它们是不是听过了卧冰求鲤的故事，在闭目养神的片刻有没有幻想过这样的大自然的赏赐，又或者其实是在进行什么鸭形阵列计算机试验（x）。
    相信这里的每个人都听过天鹅为什么会在冬天到来，大致是说天鹅在冬天会迁徙到“南方”过冬，所以会出现在这里。然而一直以来我都很疑惑，这里的冬天，河水也会结冰，草木也会枯萎，最重要的是，这里可是实打实的北方啊！如果是天鹅是从阿勒泰之类的什么地方来的话，那它们可真是懒，明明再往南飞一点就可以找到更温暖的地方，为什么要停在这呢？难道是因为喜欢吃馕？直到后来我才知道，对于天鹅们来说，这里确实算得上南方了，可能它们确实不太喜欢过于温暖的地方吧。另外，我觉得它们是真的喜欢吃馕（也可能是因为没人舍得用薯条喂它们吧）。
  • 再说循环，今天终于算是学习了，也及时写了日记。晚上回来之后看了一会Andrew Ng的DL，看了一点Pytorch，也看了一点点OpenCV。都是入门的Tutorials，不过我才知道什么Gradient Descend、Backprop都在框架里封装好了，根本不用自己去码，幸亏之前没有试着自己写。我知道这点三脚猫功夫连皮毛都算不上，更别提和大佬们比了，但是还得学，算是减小一点差距的第n阶导数吧。
  • 最后是个好玩的事。说好玩，其实应该不好玩，因为这要么是我的bug，要么是md的bug。之前的星期几，我都是用的缩写（比如## Wed），没有什么异常，可是当我写下今天的“day”的时候，也就是在文件的末尾打下## Sun准备回忆我这一天的时候，我发现它居然不见了！打开控制台看了看发现Hugo生成的并没有问题，ToC里也有Sun，可是正文中就是没有这个标题！改成Sunday就可以正常显示，Sund也可以，唯独Sun和sun不行。真是奇怪。我试着加上单引号，双引号，反斜杠，都不行。只能等明天再看看了，难道是因为现在没有太阳所以无法显示？怪啊，很怪啊。
  • 睡觉了，还有5天就回学校了。没有吵架，很难得；也有点不想回，很奇怪。

Extra#

  • 深色模式下删除线失效的问题确实是因为强制深色时一视同仁了，但是这个是内置于App里的，暂时也没想出来什么好办法，那就建议大家早点睡觉尽量在白天看吧。
  • 二级标题为Sun无法显示的问题还真是因为现在没有太阳所以无法显示，不过这个太阳是切换浅色模式的太阳，今天看到css里面样式如下 原来是Sun作为标题时的id和用来切换模式的小太阳图标的id都是sun，所以在浅色模式下就被隐藏了，最后我把图标的id改成了sun_mode，希望不会有其他的冲突。
